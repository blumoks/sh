Утилита "Параметры авторизации". Версия 1.0.0.

Функции утилиты:
 - получения информации о подключенных устройствах:
        1) Получение списка подключенных устройств: GetDeviceList;
        2) Получение информации об устройстве: GetDeviceInfo;
 - администрирования устройств:
        3) Смена пароля администратора: ChangeSoPin;
        4) Установка запрета на форматирование: Lock;
        5) Снятие запрета на форматирование: Unlock;
        6) Установка параметров авторизации: SetParams;

Получение списка подключенных устройств:
        GetDeviceList

Получение информации об устройстве:
        GetDeviceInfo [Device ID] [params]
        Параметры:
        -more                        вывод расширенной информации об устройстве.
        
        примеры вызова функции:
        GetDeviceInfo
        GetDeviceInfo -more

        !!!Важно: невозможны смена пароля администратора, установка и снятие запрета на форматирование, если устройство не было инициализировано (ни разу не были установлены параметры авторизации)!!!
        
Смена пароля администратора;
        ChangeSoPin [Device ID] [Old SO Pin] [New SO Pin]
        Device ID - идентификатор подключенного устройства (обычно - 8 цифр);
        Old SO Pin - старое значение пароля администратора;
        New SO Pin - новое значение пароля администратора;

Установка запрета на форматирование:
        Lock [Device ID] [SO Pin]
        Device ID - идентификатор подключенного устройства (обычно - 8 цифр);
        SO Pin - значение пароля администратора;

Снятие запрета на форматирование:
        Unlock [Device ID] [SO Pin]
        Device ID - идентификатор подключенного устройства (обычно - 8 цифр);
        SO Pin - значение пароля администратора;

Установка параметров авторизации:
        SetParams [Device ID] [SO Pin] [Pin Parameters] [Puk Parameters] [Other Parameters]
        Device ID - идентификатор подключенного устройства (обычно - 8 цифр); если задание параметров проводится для всех подключенных устройств, то вместо ID устройства нужно использовать флаг -all;
        SO Pin - значение пароля администратора, формат параметра: SoPin=[Pin], где Pin - собственно, пароль администратора;
        !!!Важно: если производится первое форматирование устройства, то будет установлен Pin, указанный в данном параметре!!!
        Остальные параметры не обязательно задавать; в случае, если они не будут выбраны, то будут установлены параметры по умолчанию.
        Pin Parameters - параметры Pin-кода:
                PinLen - длина Pin-кода, формат параметра: PinLen=[MinLen]<[MaxLen]; если задано только одно значение, то и минимальное, и максимальное значения PIN будут одинаковы;
                если параметр не задан, то по умолчанию устанавливается значение MinLen=7, MaxLen=10.
                PinAtt - количество попыток ввода PIN, по их истечению устройство блокируется по PIN и может быть разблокировано только вводом PUK; формат параметра: PinAtt=[Value]
                если параметр не задан, то по умолчанию устанавливается значение PinAtt=3.
                PinChar - перечень алфавитов, которые нужно обязательно использовать при задании Pin-кода:
                -d - цифры
                -u - прописные буквы
                -l - строчные буквы
                -w - спецсимволы
                Формат параметра: PinChar=[Characters]
                Пример: PinChar=-d-u-w;
        Puk Parameters - параметры Puk-кода:
                Флаг WithoutPUK: форматирование с выработкой Puk будет запрещено; в этом случае остальные параметры Puk будут проигнорированы;
                Флаг WithPUK: форматирование с выработкой Puk будет разрешено;
                В случае если ни одиин из этих флагов не выбран, считается, что форматирование с выработкой Puk разрешено.
                PukLen - длина Puk-кода, формат параметра: PukLen=[Value]; !!!Важно: длина Puk-кода должна быть четной!!!
                если параметр не задан, то по умолчанию устанавливается значение PukLen=8.
                PukAtt - количество попыток ввода PUK, по их истечению устройство блокируется по PUK; формат параметра: PukAtt=[Value]
                если параметр не задан, то по умолчанию устанавливается значение PukAtt=3.
        Other Parameters:
                Флаг MakeConstant - нужен, чтобы "заморозить" параметры.
                !!!Важно: после заморозки параметров авторизации, отменить это действие будет невозможно!!!

ShAuthParams.exe SetParams 35010542 SoPassword=1234567 WithoutPUK PukLen=2 PinAtt=5 PinChar=-d-u-w-l 

                
Примеры вызова функций:
        ChangeSoPin 00000002 123123123 asdfqwerty
        Lock 00000002 123123123
        Unlock 00000002 123123123
        SetParams 00000002 SoPin=123123123
        SetParams 00001617 SoPin=1234567 WithoutPuk PinLen=7 PinAtt=5 PinChar=-d-u-w
        SetParams -all SoPin=1234567 WithPuk PukLen=12 PukAtt=5 PinLen=10<16 MakeConstant
